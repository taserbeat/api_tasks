# api_tasks
タスク管理アプリ

- サーバーサイド  
Django RestFramework

- フロント  
React + redux toolkit


## セットアップ

```shell
# クローン
git clone https://github.com/taserbeat/api_tasks.git

# サーバーサイド
cd api_tasks
python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
python manage.py runserver


# フロント
cd rtk_tasks
yarn install
yarn start

```
